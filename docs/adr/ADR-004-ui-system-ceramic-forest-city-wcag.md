# ADR-004 — UI System: Ceramic/Forest City con WCAG 2.2 AA

## Contexto

Se busca una identidad visual consistente y premium sin comprometer legibilidad ni accesibilidad.

## Decisión

Adoptar lineamientos:

- Ceramic Glassmorphism selectivo (HUD/overlays),
- Forest City como atmósfera de fondo,
- accesibilidad WCAG 2.2 AA como restricción de diseño.

## Consecuencias

- Mejora coherencia visual entre frontends.
- Reduce “UI improvisada” en nuevas pantallas.
- Obliga a validar contraste, foco y target size en QA.

## Estado

Aceptado.
