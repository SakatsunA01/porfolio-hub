# Frontend (Vue 3 + Vite)

## Requisitos
- Node.js 20+
- npm 10+

## Instalacion
```bash
npm install
cp .env.example .env
npm run dev
```

Frontend en: `http://127.0.0.1:5173`

La API se toma desde `VITE_API_URL`.
